Top Dungeon

Top Dungeon is a dungeon crawler game developed as a group project by me and two other students during our second year of Computer Science degree. The game was created using Unity, C# and Visual Studio as the primary IDE, with gameplay inspired by classic dungeon crawler titles.

The project was developed using pre-made assets from itch.io, and coded in C#.

My contributions: Player movement, Enemy, UI, Dungeon Desing, QA, Combat system.
